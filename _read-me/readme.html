Home v1 background Image

To set the background image of Header section you'll have to open the associated file (e.g. home-v1.html) and find the following line(s) of code there::

                <div class="hero v2 section-padding" 
                style="background-image: url(images/header/hero_1.jpg)">

Now just change the file reference in "background-image :" property to your desired image url.

Overlay

You can adjust the opacity of overlay section to achieve proper conrast. To do that find the following code and change the number trailing the 'op' class to any odd number between 1-9.

<div class="overlay op-3"></div>

Map Location

To set the location in google map you'll have to maps.js file and find the following line(s) of code there:


            'Marker': [
	            {
	                location_latitude: 28.550913,
	                location_longitude: -81.456819,
	                type_point: 'For Rent',
	                bed_num: '4',
	                bath_num: '3',
	                garage_num: '2',
	                total_area: '2240',
	                map_image_url: 'images/property/property_1.jpg',
	                price: '15000',
	                name_point: 'Villa on Hartfold',
	                url_point: 'single-listing-one.html',
			},

Now change the latitude & logtitude vale to your desired loacation's latitude & longtitude.

To set the location in Leaflet map you'll have to leaflet-custom.js file and find the following line(s) of code there:


            [locationData('single-listing-one.html', 
            'images/property/property_1.jpg',
             "Villa on Hartfold", '15000',
              'For Rent', '4', '3','2','2240'),
               28.550913, -81.456819],

Here last second value is latitude and the last one is longtitude value.Now change the latitude & logtitude vale to your desired loacation's latitude & longtitude.

Counter Up

To set the Counter up you'll have to open the associated file (e.g. about.html) and find the following line(s) of code there:


            <h6 class="counter-value" data-from="0" data-to="10" data-speed="2500">

Now change the "data-from" & "data-to" value to your desired number.
You can also control the speed of counter animation by "data-speed" value.

Contact form setup

To setup the contact form, open "send_mail.php file in the "php" folder and find the following lines of code there:

$toEmail = "hello@yourdomain.com";
 
Just replace it with the email ID where you would like the contact form emails to be sent to.
